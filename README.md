# SCI Editor (科学论文辅助编辑工具)

SCI Editor 是一款专为生物医学科学论文设计的自动化编辑与格式校对工具。它能够识别常见的格式错误、缩写不规范、统计结果格式问题等，并提供智能化的一键修复功能。

## 主要功能

- **自动化解析**：精准识别 .docx 文件的结构，包括标题、作者、单位、摘要及正文各部分。
- **中国作者姓名规范**：自动识别连接符姓名（如 `Tai-xian`）并规范为标准格式（`Tai-Xian`）。
- **短标题 (Running Title) 自动生成**：提取第一作者姓氏自动补全短标题，并正确处理 `et al.` 的斜体格式。
- **缩略词一致性增强 (Rule 3.10)**：
  - **动态阈值**：仅在术语出现 $\ge 3$ 次时启用缩写。
  - **首词定义**：自动确保术语在区段首次出现时附带定义，后续保持缩写。
  - **低频词清理**：自动将出现频率极低的缩写还原为全称，提升论文正式感。
- **统计格式规范 (CI/OR)**：标准化 `OR = 0.85 (95%CI: 0.7-1.2, P = 0.01)` 等复杂统计结果的格式。
- **摘要结构检查**：针对 Original Article 等不同稿件模式，自动核对摘要必备结构（Background, Aim, Methods 等）。
- **图形化报告**：生成详细的 HTML 修复报告，方便用户核对修改点。

## 安装与使用

### 运行环境
- Windows 操作系统
- 已安装 Python 3.10+ (可选，若直接运行 .exe 则不需要)

### 直接使用
1. 下载 `dist/sci_editor.exe`。
2. 运行程序，在 GUI 界面选择需要检查的 .docx 文件。
3. 点击“执行检查”，查看报告并选择“应用修复”。

### 开发者模式
```bash
# 克隆仓库
git clone https://github.com/Jiangsir0113/sci_editor.git
cd sci_editor

# 安装依赖
pip install python-docx jinja2 pyinstaller

# 运行程序
python sci_editor/main.py
```
